# Intelligent Medicine Reminder and Medication Tracking Platform

A healthcare web platform that helps users manage medicines, track doses,
view medication schedules, and receive refill reminders.

## Milestone 1 Features

- User registration and login
- JWT authentication
- Role-based access control design
- PostgreSQL database setup
- User profile management API
- Dashboard UI
- Healthcare workflow documentation
- Database schema documentation
- UI wireframes

## Tech Stack

- React + Vite
- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT Authentication
- GitHub
