# Intelligent Medicine Reminder and Medication Tracking Platform

## Problem Statement

Patients may forget to take medicines, miss doses, lose prescriptions,
or fail to refill medicines before they run out. This platform helps users
track medicines, receive reminders, record doses, and monitor adherence.

## Users

1. Patient/User
2. Caregiver (future feature)
3. Admin (future feature)

## Milestone 1 Scope

- User registration and login
- JWT authentication
- Role-based access control
- User profile management
- PostgreSQL database setup
- Dashboard UI
- Basic medicine summary using sample data

## Future Scope

- Add medicines
- Reminder notifications
- Dose tracking
- Missed-dose alerts
- Refill alerts
- OCR prescription scanning
- Caregiver dashboard
